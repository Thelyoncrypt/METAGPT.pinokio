# MetaGPT Installation Script for Pinokio

This repository contains a one-click installation script for MetaGPT, designed to be run in Pinokio.

## Usage

1. Download the `install_MetaGPT.json` file from this repository.
2. Upload the file to Pinokio using the upload or import option.
3. Run the script in Pinokio to clone the MetaGPT repository, create and activate a virtual environment, and install MetaGPT dependencies.
4. Enjoy using MetaGPT to create programs with a single prompt!

## Requirements

- Pinokio environment
- Python 3.x
- Git
